HTML Reference Sheet

How to use HTML tags

Tag Name Description
h1 Heading Level 1 The largest heading. There is usually only one used on each page.
br Line Break This tag is used to create a ne line.
p Paragraph This tag is used to create a new paragraph.
strong Strong This tag is used to make your text bold. Place the text you want to be bold in between the opening and closing tags.
em Emphasized Text Place text that you want emphasized inbetween this text. Your words will look italicized
